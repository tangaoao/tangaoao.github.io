项目搭建

下载Vue以及Vue脚手架（Vue-cli)
  1. 下载Node.js,使用npm进行下载
  2. 在cmd中输入：npm install -g vue【下载Vue】
  3. 输入：npm install @vue/cli -g 【下载脚手架】
  4. vue -V 【查看vue版本】
创建Vue项目
  1. 创建项目目录：SpringBoot-Vue
  2. 在项目目录下打开cmd
  3. 输入：vue creat [项目名]
    1. 选择Manually select falures,点击回车
    2. 选择Vuex、Babel、TypeScript和Router，回车
    3. 选择Vue版本 Vue3.x,回车
    4. 输入y,选择In Package.json，输入N
  4. 进入项目文件夹：cd [文件夹名称]
  5. 运行项目：npm run server
  6. 结束项目：Ctrl +C
下载使用Element–Plus【桌面端组件库】
  1. 命令框输入：npm install element-plus –save -g
